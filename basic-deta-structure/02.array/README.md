# Array

### 配列の仕組み
- 同じデータ型となる要素を集めたもの
- 要素を指定するための通し番号は添字(index)

### Pythonと配列
- Pythonのリスト (List)
  - 一般的な言語における配列とほぼ同じものとする文献が多い
  - CPythonなどの実装では可変長配列

- Pythonの配列 (array)
  - arrayモジュールをimportして使用
  - 全ての要素が同じ型
  - リストと比較して、高速でメモリー節約できる

- NumPyのn次元配列 (ndarray)
  - 外部のパッケージ (https://numpy.org/)
  - ベクトルや行列などの数値演算
